SUPPORT & TROUBLESHOOTING
================================================================================

Common Issues:

1. Chat endpoint returns 500
   Solution: Check if LLM provider is configured correctly
   Fallback: Use standalone reasoning (LLM_PROVIDER=none)

2. Search returns no results
   Solution: Try broader search term
   Alternative: Browse by act or category

3. Analysis takes long time
   Solution: Expected for first run (loads model)
   Cache: Subsequent calls are faster

4. Document upload fails
   Solution: Check file format (PDF, DOCX, TXT, images)
   Size: Keep documents under 50MB

5. Report generation error
   Solution: Ensure case has analysis data
   Try: Regenerate case AI summary first
